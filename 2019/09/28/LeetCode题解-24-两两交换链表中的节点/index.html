LeetCode题解-24. 两两交换链表中的节点

发表于 2019-09-28 分类于 LeetCode

题目描述

给定一个链表，两两交换其中相邻的节点，并返回交换后的链表。

你不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。

示例:

给定 1->2->3->4, 你应该返回 2->1->4->3.

思路

两两交换，然后相连，画一遍图熟悉一下交换过程就可以开始敲了，使用哑节点更方便。
